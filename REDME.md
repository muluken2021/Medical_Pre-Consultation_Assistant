Project: Medical Pre-Consultation Assistant

# Problem Description

problem : Ethiopian public hospitals face extreme patient volumes, giving doctors < 5 minutes per patient.

The Barrier: Language diversity (Amharic, Afaan Oromo, etc.) vs. Clinical Documentation (strictly English) creates a "translation lag" during the visit.

The Impact: Doctors spend 60% of the visit on basic data entry ("When did the pain start?") rather than clinical decision-making, leading to rushed diagnoses.







# Solution Description

The Agent: A Goal-Based AI Agent that conducts an automated "Intake Interview" in the waiting room.

The Mechanism: Using Natural Language Processing (NLP), the agent interviews the patient in their native tongue and uses a "Decision Tree" logic based on Ethiopian Standard Treatment Guidelines (STG).



The Goal: Generate a structured SBAR (Situation, Background, Assessment, Recommendation) report in English for the doctor to review in under 30 seconds.
